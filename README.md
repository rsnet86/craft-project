# Craft-Project

Demonstrate the design of tinyurl by using public cloud and microservices(EKS). 

Accomplished the project using the below services:
1- Redis in ec2
3- EKS "k8s"
4- RDS "postgres"
5- ECR
6- URL shortener in golang
